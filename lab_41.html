Lab 41: Fetching Changes

Goals

  • Learn how to pull changes from a remote repository.

Steps

Execute:

cd ../cloned_hello
git fetch

Output:

$ git fetch
From /Users/kenbarker/projects/edgecase/goodies/git_immersion/auto/hello
   ca1ff7f..fb5859f  master     -> origin/master

At this point the repository has all the commits from the original repository. However, they have not been merged into the local master branch.

Check the README

Execute:

cat README

Output:

$ cat README
This is the Hello World example from the git tutorial.

See, no changes.
