Lab 21: More Structure

Goals

  • Add another file to our repository

Steps

Now add a Rakefile

Let’s add a rakefile to our repository. The following one will do nicely.

File: Rakefile

#!/usr/bin/ruby -wKU

task :default => :run

task :run do
  require 'lib/hello'
end

Add and commit the change.

Execute:

git add .
git commit -m 'Added a Rakefile.'

You should be able to use Rake to run your hello program now.

Execute:

rake

Output:

$ rake
(in /Users/kenbarker/projects/edgecase/goodies/git_immersion/auto/hello)
Hello, World!
