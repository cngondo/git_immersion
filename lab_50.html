Lab 50: Pushing a Change

Edit the README and commit it

File: README

This is the Hello World example from the git tutorial.
(Changed in the original and pushed to shared)

Execute:

git checkout master
git add README
git commit -m 'Added shared comment to readme'

Now push the change to the shared repo.

Execute:

git push shared master

Output:

$ git push shared master
To ../hello.git
   745ca58..1541d06  master -> master

NOTE: We had to explicitly name the branch master that we were pushing. It is possible to set it up automatically, but I never remember the commands to do that. Check out the “Git Remote Branch” gem for easy management of remote branches.
